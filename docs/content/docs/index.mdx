`obsidian-dev-utils` is a collection of essential functions and configurations designed to streamline
the development of Obsidian plugins. It is published as a dual-format (ESM + CJS) npm package.

This site hosts the hand-written guides and the full **API reference** generated from the library's
TSDoc. Browse the API in the sidebar, or start with a guide below.

## Guides

- [Commands](./guides/cli-commands.md) — CLI commands exposed by the library.
- [Helper Functions](./guides/helper-functions.md) — general-purpose and Obsidian-specific helpers.
- [Styling](./guides/styling.md) — extensible styles for plugin UI controls.
- [Setting Components](./guides/setting-components.md) — ready-made settings-tab components.
- [Modals](./guides/modals.md) — alert / confirm / prompt / select modals.
- [Debugging](./guides/debugging.md) — enabling and using debug output.
- [Plugin Helpers](./guides/plugin-helpers.md) — helpers to simplify your own plugin.
- [Demo Vault](./guides/demo-vault.md) — ship and open a demo vault.
- [Testing](./guides/testing.md) — test setup and utilities.
